# IPL Data Analysis and Visualization

## Project Overview

This project analyzes Indian Premier League (IPL) match data using Python, Pandas, and Matplotlib.

The objective is to perform Exploratory Data Analysis (EDA) and create meaningful visualizations to understand team performance, player achievements, venue statistics, and match trends.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Features

- Data Cleaning
- Exploratory Data Analysis
- Team-wise Performance Analysis
- Player of the Match Analysis
- Venue Analysis
- City-wise Match Analysis
- Toss Decision Analysis
- Win Margin Analysis

## Visualizations

- Matches Played Per Season
- Team-wise Wins
- Toss Decision Distribution
- Top Player of the Match Winners
- Top Cities Hosting Matches
- Top Venues
- Average Runs by Teams
- Win Margin Analysis
